3.2安装

请按下列步骤安装SiteView ECC 8.1.3：

—  安装说明

安装之前,请确认系统补丁安装情况, Windows 2003+SP1或SP2, Windows XP+SP2或SP3

—  安装过程

执行WebECC 8.1.3 Setup.exe，进入WebECC8.1.3的正式安装，如下图。

图3.2-1

单击[下一步]，打开窗口，如下图。

图3.2-2

用户需仔细阅读这份软件许可协议，如无异议，选择“我接受许可证协议中的条款”，单击[下一步]，打开窗口，如下图。

图3.2-3

这一步是要求用户选择安装程序的文件夹，默认安装在C:\SiteView\WebECC下；如果需要安装到其它文件夹，单击[浏览]，用户可在打开的Windows文件夹中选择或者创建一个文件夹。但是安装路径中不能包含空格和中文名称。单击[下一步]，打开窗口，见下图。

图3.2-4

选择需要安装的组件，确认后，单击[下一步]，打开窗口，见下图。

图3.2-5

配置WEBECC访问的端口号和访问的服务器IP，用户可以根据需要进行设置，也可以采用默认设置。假如80端口没有被占用，建议您使用8080端口，默认为8181端口。确认后，单击[下一步]，继续安装，见下图。

图3.2-6

当前步骤，用来进行配置信息的确认检查，如果需要修改，单击[上一步]进行修改。否则单击[下一步]开始进行安装。见下图。

图3.2-7

接着弹出提示，进行服务的安装。见下图。

图3.2-8

服务安装完成后，在经过几秒的等待之后，安装程序弹出窗口，提示安装结束。见下图。

图3.2-9

    当进行到这一步的时候，恭喜您，您已经成功的在计算机中安装了WEBECC。并且此页面中默认选中运行WebECC 8.1.3,如果采用此设置,单击[完成]，则立即弹出WEBECC登陆界面，直接可以进行登陆，见下图。

图3.2-10

WebEcc 8.1.3安装完成，安装程序已经在计算机中安装了六项服务，并且全部自动启动。

1.     SiteView DB，在任务管理器中进程是svdb.exe。

2.     SiteView Monitor Control，在任务管理器中进程是MonitorContrl.exe和MonitorSchedule.exe。

3.     SiteView Receive Service，在任务管理器中进程是SiteViewReceive.exe。

4.     SiteView Web Server。

5.     SiteViewEcc813WEB，在任务管理器中进程是tomcat.exe。

6.     TomcatMonitorService，在任务管理器中进程是TomcatMonitorService.exe。

这六项服务处于“已启动”状态，确保了WebEcc8.1.3正常运行。

安装完成后桌面会生成SiteViewEcc8.1.3快捷方式图标，可以双击打开登陆。

从上图中的浏览器地址可见到，访问WebEcc8.1.3的地址根据您的定义自动生成DD“http://ip:port/ecc”，例如“http://localhost:8181/ecc”或者“http://127.0.0.1：8181/ecc”。在上图中输入登录名和密码，单击登录，即可进入WebEcc8.1.3的主界面。从别的服务器访问WEBECC，只需要把配置的“http://ip:port/ecc”中的ip，换成服务器的IP地址即可。

注意事项：

1．如果该机器以前安装过SiteView ECC软件，请将旧版本卸载，然后再安装此版本，切记不要先安装新版本再卸载旧版本。

2．安装完成后，切记要重新启动再使用本软件。

3．浏览器必须使用IE7.0以上版本。

4．在安装的过程中目录文件名不支持长路径名和中文名称。
