8.7.1 Exchange

Microsoft Exchange Server 是一个主要的Intranet协作应用服务器，适合有各种协作需求的用户使用。Exchange Server协作应用的出发点是业界领先的消息交换基础，它提供了业界最强的扩展性、可靠性和安全性和最高的处理性能。Exchange Server 是一个设计完善的邮件服务器产品，提供了通常所需要的全部邮件服务功能。除了常规的SMTP/POP3协议服务之外，它还支持IMAP4、LDAP和NNTP协议。
